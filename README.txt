Face mask detection that detects when a person wears a mask correctly, does not wear a mask and wears it incorrectly.

It gives a real information to the admin alerting when a person does not wear a face mask

You need to install the following using the commands below

For Example:
pip install numpy (Do this for every listed libraries)

numpy
pillow
torch
tqdm
terminaltables
torchvision
matplotlib
tensorboard
opencv-python

TO RUN THE DETECTION
Use the following command:
python video.py --weights_path checkpoints/yolov3_ckpt_35.pth
